## My Portfolio

### Features:
- About Me
- Skills and Technologies
- Experience
- Resume
- Navigation bar

### Prerequisites:
Node.js 20.12.0 or later installed.

### Installation
#### 1. Clone the repository
```
git clone https://github.com/ashiq012/portfolio
```
#### 2. Install dependencies
```
npm install
```
#### 3. Start the website (locally)
```
npm run dev
```
